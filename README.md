# Project Description
Aplikasi ini merupakan aplikasi yang dibuat untuk memenuhi tugas dari Blue Bird Frontend Challenge. Aplikasi ini dibuat menggunakan ReactJS (dengan Framework Next.JS) dan menggunakan API dari Bluebird.
